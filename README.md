# India-General-Election-Data-Visualization-
Power BI Dashboard analyzing India General Election 2024 with insights on vote share, seat distribution, and party performance
Dashboard
📌Project Overview
This project presents an interactive Power BI dashboard analyzing the results of the India General Elections 2024. The dashboard provides a comprehensive view of election outcomes at national, state, and constituency levels.
It highlights party performance, seat distribution, vote share, and strike rates using visually engaging charts and KPIs. The goal of this project is to transform raw election data into meaningful insights for better understanding and analysis.

🎯 Objective
To analyze election results in a structured and visual format
To compare the performance of major alliances and political parties
To identify winning trends at state and constituency levels
To evaluate party efficiency using strike rate and vote share
To build an interactive dashboard for easy data exploration

Dashboard Analysis
The dashboard is divided into four major sections: Overview Analysis, State Analysis, Constituency Analysis, and Party Strategy & Performance Analysis. Each section provides a different level of insight into the India General Election Results 2024.

🧾 1. Overview Analysis
This section provides a national-level summary of the election results.

🔹 Key Metrics:
Total Seats: 543
NDA Alliance: 292 seats (54%)
INDIA Alliance: 234 seats (43%)
Others: 17 seats (3%)
🔍 Insights:
The NDA alliance holds the majority and forms the government
INDIA alliance shows strong competition
Smaller parties contribute to coalition dynamics
🎯 Purpose:
Quick understanding of overall election outcome
High-level comparison between major alliances

🛠️Tools & Technologies Used
Power BI :- Used for building interactive dashboards, data visualization, and report creation.
Microsoft Excel :- Used as the primary data source for storing and cleaning election data.
DAX (Data Analysis Expressions) :- Used for creating calculated measures such as:
Vote Share (%)
Strike Rate (%)
Total Votes
KPI metrics
Data Modeling :- Relationships were created between tables (State, Constituency, Party) to enable accurate filtering and analysis.

🎨Design Approach
Structured layout (Overview → State → Constituency → Performance)
Consistent color coding for alliances
KPI cards for quick insights
Interactive slicers (State & Constituency)
Multiple visuals (bar, pie, scatter, table) for better storytelling
Focus on clean and readable UI
